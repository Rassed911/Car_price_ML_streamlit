# 🚗 Калькулятор стоимости подержанных автомобилей (Machine Learning + Streamlit)

### 📌 Описание проекта
В этом проекте я разработал интерактивное веб-приложение, которое предсказывает рыночную стоимость подержанного автомобиля на основе его характеристик. Проект охватывает полный цикл: от предобработки данных и обучения модели до развертывания удобного интерфейса.

### 🛠️ Стек технологий:
* **Язык программирования:** Python
* **Библиотеки ML:** Scikit-Learn (Random Forest Regressor)
* **Обработка данных:** Pandas (One-Hot Encoding, Preprocessing)
* **Веб-интерфейс:** Streamlit (Deployment)

### 🚀 Ключевые этапы разработки:
1. **Подготовка данных:** 
   * Использовал датасет реальных объявлений о продаже авто (150 записей).
   * Реализовал **One-Hot Encoding** для категориальных признаков: модель автомобиля (`SE`, `SEL`, `SES`), цвет кузова и тип трансмиссии.
2. **Машинное обучение:** 
   * В качестве основного алгоритма выбрал **Random Forest Regressor**.
   * Оптимизировал параметры модели, добившись средней ошибки (MAE) в районе **$1178**, что является отличным показателем точности для данного объема данных.
3. **Разработка интерфейса (UI):** 
   * Создал веб-приложение на базе **Streamlit**.
   * Добавил интерактивные элементы: слайдеры для выбора года выпуска и пробега, а также выпадающие списки для выбора комплектации и цвета.

### 📈 Результаты:
Модель успешно учитывает нелинейные зависимости (например, влияние пробега на цену при разных типах трансмиссии) и выдает мгновенный прогноз стоимости. 

**Пример прогноза:** Для автомобиля 2012 года выпуска в комплектации SES с пробегом 30,000 миль модель предсказывает стоимость около **$15,000**.

### 🔧 Как запустить проект локально:
1. Клонируйте репозиторий.
2. Установите необходимые библиотеки: `pip install pandas scikit-learn streamlit`
3. Запустите приложение: `streamlit run used_car_app.py`
4. Ссылка на датасет: `https://github.com/stedy/Machine-Learning-with-R-datasets/blob/master/usedcars.csv`
